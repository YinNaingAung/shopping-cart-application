# Mini Shopping Cart
This project is a mini shopping cart application built with vanilla JavaScript. It allows users to browse products, add items to the cart, update quantities, and remove items. The cart data is stored in localStorage, so it persists even after refreshing the page.

## What I learned

- DOM manipulation
- Event handling
- State management logic
- Dynamic UI rendering
- Working with arrays (map, filter, find)
- Using localStorage for data persistence
